# ZeinebVizLib

Une bibliothèque personnelle de visualisation en Python utilisant `matplotlib`.

## Description

ZeinebVizLib permet de créer facilement des graphiques stylés avec un style visuel cohérent.  
Elle inclut 5 types de visualisations minimum :  
- `styled_line` : courbes (line plot)  
- `styled_bar` : histogrammes en barres (bar plot)  
- `styled_scatter` : nuage de points (scatter plot)  
- `styled_hist` : histogrammes (hist plot)  
- `styled_box` : diagrammes en boîte (boxplot)  

Chaque graphique utilise un style global avec des couleurs, tailles de polices et axes harmonisés.
